Bootloader

The bootloader is stored on a small 4Mbit NOR FLASH chip (XMC 25QH40BZ1G)/XM25QH40

The firmware size is limited to 15MB

I. Boot process

1. The firmware is read to address 0xa0000200

2. Main then checks for the 512 byte header to start with magic number 0x4746434c (LCFG)

3. Main copies this 512 bytes header to 0xa0f00000

4. shifts the length read from 0xa0000400 to 0xa0000200

5. calculates and checks CRC32 at offsets 0x184 and 0x18C

6. jumps to 0x80000200 (bisrv.asd load address and entry point)
